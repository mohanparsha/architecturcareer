1
Foundation Building
Months 1–2
Linux OS — Ubuntu 24.04, filesystem, permissions, users, processes, systemd, package management
Shell Scripting — Bash fundamentals, variables, loops, conditionals, cron, text processing (grep, awk, sed)
Python (primary) + Go intro — Scripting, file I/O, API calls, JSON/YAML parsing, automation scripts
Git & GitHub — Branching, merging, rebasing, PRs, conflict resolution, Git workflows
◆ Automate server setup with a Bash script + push to GitHub
2
Core DevOps Skills
Months 3–4
Networking — OSI model, TCP/IP, DNS, IPv4/IPv6, CIDR, firewalls, load balancing, proxy/reverse proxy
Web Servers — Nginx & Apache — config, virtual hosts, reverse proxy, SSL/TLS termination
Advanced Bash — Functions, arrays, trap, heredoc, process substitution
YAML & JSON — Syntax mastery, jq parsing, config file fluency
◆ Deploy a static site on Nginx with SSL behind a reverse proxy
3
Containerization & Cloud
Months 5–6
Docker — Images, containers, Dockerfiles, volumes, networks, Docker Compose, multi-stage builds
Container Registries — Docker Hub, AWS ECR, GHCR, Google Artifact Registry
Kubernetes — Pods, Deployments, Services, Ingress, ConfigMaps, Secrets, RBAC, Helm charts
Cloud (AWS) — EC2, S3, VPC, IAM, RDS, Route 53, CloudWatch, free tier labs
◆ Deploy a multi-container app on Kubernetes with Helm
4
CI/CD & Infrastructure as Code
Months 7–8
CI/CD Pipelines — Jenkins, GitHub Actions — build, test, deploy automation, webhook triggers
Terraform — HCL, providers, state, modules, plan/apply workflow (note: OpenTofu as emerging fork)
Ansible — Playbooks, roles, inventory, idempotency, configuration management at scale
GitOps — ArgoCD / Flux — declarative deployments, Git as single source of truth
◆ Build a CI/CD pipeline: code → build → test → deploy to K8s
5
Monitoring, Security & DevSecOps
Months 9–10
Observability Stack — Prometheus + Grafana + Loki — metrics, dashboards, log aggregation, alerting
SRE Practices — SLIs, SLOs, SLAs, error budgets, incident response, post-mortems
DevSecOps — SAST (Semgrep, SonarQube), DAST (OWASP ZAP), container scanning (Trivy), OWASP Top 10
Secrets & IaC Security — HashiCorp Vault, Checkov/Trivy for IaC, Gitleaks for secret detection
◆ Monitor a K8s cluster with Prometheus/Grafana + integrate security scanning in CI
6
Real-World Projects & Portfolio
Months 11–12
Capstone Project 1 — Full CI/CD: GitHub → Jenkins → Docker → K8s with monitoring and rollback
Capstone Project 2 — GitOps: ArgoCD/Flux + Terraform IaC + security pipeline integration
Interview Prep — Linux, Docker, K8s, CI/CD, Cloud, Terraform, Networking, Security scenarios
Portfolio & Soft Skills — GitHub portfolio, resume, LinkedIn, communication, collaboration, documentation
◆ Job-ready portfolio with 2–3 comprehensive DevOps projects on GitHub
Recommended Certifications — pursue after completing the roadmap to validate your skills
